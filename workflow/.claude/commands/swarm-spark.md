Read the project adapter at `.claude/adapter.md` for test commands, conventions, and the Spark agent profile config, then read and follow the swarm-spark skill at `.claude/skills/swarm-spark/SKILL.md`.

Use $ARGUMENTS as the plan file path. Optional: append specific task IDs to run a subset.
